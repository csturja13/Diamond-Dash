# Diamond-Dash
Catch the diamond with the plate

**you would need "OpenGL library" to run the .py file.
 you can download and install "OpenGL library" via VSCode terminal commands.

 Game rules-->
--> left & right arrow keys to move the plate
--> diamond falling speed will increase after catching each diamond
--> click the back arrow button to start a new game

#MPL(midpoint line drawing algorithm) along with 8 way symmetry is used in this game.
